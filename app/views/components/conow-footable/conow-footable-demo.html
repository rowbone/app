组件描述
封装后的响应式表格使用只需要在所开发的页面中使用到一条指令即可。
示例如下：
                            
                              <conow-footable  search-name="姓名" ft-result-array="result" ft-top-title="带标题不带超链接的" url="/service/notice!queryAllNoticeList" ft-titile="标题;来源;创建者:phone;操作:phone"  ft-search="search" ft-item="TITLE;SOURCE;CONTRIBUTOR;save():btn-info:保存">
                      <a class="btn btn-info">aaaa</a>
                    </conow-footable>
                           
                           需导入以下文件:
                          .state('app.table.conowFootable', {
                  url: '/conowFootable',
                  templateUrl: 'other/demo/conow-footable/conowFootableDemo.html',
                  resolve: {
                      deps: ['uiLoad',
                        function( uiLoad){
                          return uiLoad.load([
                                              'vendor/modules/conow-footable/directive/conow-footable.js',
                                              'vendor/plugins/jquery/footable/footable.core.css',
                                              ]);
                      }]
                  }
              })

url(必填)
例：
生成表格内容数据的请求地址
search-name
例：
自动生成搜索栏里的label名
no-search
例：
是否启用搜索栏，若为true则不启用 默认不写为启用
ft-top-title
例：
标题,不写则不带标题
ft-titile(必填)
例：
标题名称

ft-item(必填)
例：
每一单元格的属性

ft-search(必填)
例：
搜索条件对象

ft-result-array(必填)
例：
远程数据返回的结果集

row-link
例：
点击跳转的超链接

pagesize
例：
每页显示条数：默认6

maxsize
例：
最多显示多少页：默认6

no-data-tip
例：
无数据情况下的提示信息：默认“暂无数据”

show-light
例：
点击尚亮
