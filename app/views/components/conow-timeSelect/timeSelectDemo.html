组件描述
可选属性
   	日期组件
{
    start-date(必填)    : "双向数据绑定的开始时间"
    end-date(必填)      : "双向数据绑定的结束时间"
    date-init(非必填)    : "设置一开始选中的天数：YYYY-MM-DD"
    comf-btn(非必填)      ："为true则显示确定按钮"
	single-time(是否单选 非必填):"为true则只需填写一个"
	format(非必须)       :'设置返回的时间格式' eg.YYYY-MM-DD
	min-date(非必须)	   :最小日期,格式为字符串eg.min-date="'2015-05-25'"
	max-date(非必须)	   :最大日期,格式为字符串(同上)
	half-day-choose="true"(非必填) 设置为true时，并且有确定按钮时，才可以选择全天、上午、下午。
	choose-type(非必填)       : "双向数据绑定的全天、上午、下午"
}

 	时间组件
{
    time(必填)    : "双向数据绑定的时间:格式可以为hh:mm ,YYYY-MM-DD hh:mm:ss,YYYY-MM-DD等"
    end-date(必填)      : "双向数据绑定的结束时间"
    date-init(非必填)    : "若为true则每次点击组件不带原时间,默认为false,带原时间"
    format(非必须)       :'设置返回的时间格式' eg.YYYY-MM-DD
	ex-date(非必须)      :'可以使用他来添加前面日期'
	min-time(非必须)	   :设置最小时间，格式为字符串 eg.min-time="'20:12'"
	max-time(非必须)	   :设置最大时间，格式为字符串 (同上)
}
