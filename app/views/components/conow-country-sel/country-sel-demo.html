学校字母分组选择
{{ vm | json }}
字母分组使用说明
  1. 路由跳转中引入文件：'views/components/conow-country-sel/js/conow-alphabet-group-sel.js'
  2. 初始化指令：<input type="text" class="form-control" conow-alphabet-group-sel="对象参数" ng-model="vm.selected">
  3. 对象参数详细说明
    参数名称 参数说明 默认值 适用场景
    dataSrcUrl 初始化时的数据接口地址 - 全部数据加载或者只加载COMMON数据
    getSelectedValueUrl 根据code转换获取value的数据接口地址 - 部分数据加载时使用
    getDataInitialsUrl 点击单个字母时获取对象选项的接口地址 - 部分数据加载时使用
    searchUrl 搜索对应的数据接口地址 - 部分数据加载时使用
    isShowSearch 是否显示搜索框 通用
    isLoadingAll 是否加载所有 - 通用
    isHasCommon 是否包含"热门" true 通用
    isCommonExpanded "热门"是否默认展开 true 通用
    isShowAllLabels 是否显示所有字母 false 通用
    isMultiSelect 是否多选
    selectKey code 对应的数据字段 CODE 通用
    selectValue value 对应的数据字段 VALUE 通用
    selectTitle title 对应的数据字段 VALUE 通用
    colsPerRow 待选项每一行显示的个数 3 通用
  4. 接口字段说明
    接口名称 参考
    dataSrcUrl '/service/school!queryIsCommSchool'(部分加载) '/service/country!queryAllCountry'(全部加载)
    getSelectedValueUrl '/service/school!queryByCode'
    getDataInitialsUrl '/service/school!querySchoolByInitials'
    searchUrl '/service/school!querySchoolByNameSpell'
