通用级联选择
{{ entity | json }}
使用说明
  1. 调用指令：conow-cascade-select
  2. 示例 
    <input type="text" class="form-control" conow-cascade-select sel="sel" titles="titles" url="'/service/common!queryOptions?type=DICT_LEVEL&DICT_CODE=HR_RETIRED_ARMY_RANK'" ng-model="entity.sel">
  3. 属性说明：
    • sel：已选项
    • titles：选择页签标题
    • url：读取数据的后台接口
功能扩展
  1. 根据参数确定选择的层级数
  2. 不能选择的层级隐藏或者disabled掉
