使用说明
  1. 使用前先引入ng-table.js和ng-table.css
  2. table元素添加ng-table属性，属性值是对应的配置参数
  3. controller中添加配置属性：$scope.tableParams = new ngTableParams(parameters, settings)
  4. 配置参数说明：
    参数对象 参数名称 说明
    parameters page 当前显示的页码
    count 每页显示的数据行数
    filter 定义过滤器
    sorting 定义排序
    settings total 表格显示的总数据条数
    counts
    defaultSort
    groupBy
    getData
