使用说明
  1. 引入文件 object-table.js & object-table.css
  2. 调用指令 object-table
  3. 参数列表
    参数名称 是否必填 默认值 说明
    from-url 是 - 后台数据接口
    data 是 - json数据
    display 是 - 单页显示记录数
    headers 是 - 表头显示内容，用,分隔
    search 否 true 是否显示搜索
    select 否 null 是否可选/单选/多选
