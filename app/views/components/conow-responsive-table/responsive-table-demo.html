使用说明
  1. 引入文件 conow-datatable.js & conow-datatable.css
  2. 调用指令 conow-datatable
  3. 参数列表
    参数名称 是否必填 默认值 说明
    from-url 是 - 后台数据接口
    json-data 是 - json数据
    display 是 - 单页显示记录数
    headers 是 - 表头显示内容，用,分隔
    search 否 true 是否显示搜索
    select 否 null 是否可选/单选/多选
    tableInstance 否 对应的table的实例，提供一些操作方法
    class 否 作用于 conow-datatable 的class。例如 horizontal-scroll 可用于生成水平滚动的表格
  4. tableInstance 说明:table的实例，用于在controller中对table进行操作
    方法名 说明
    reload 重新加载
  5. table 实例中可以访问业务 controller 中的属性和方法。$owner 即为业务controller的引用。
