核心

需了解AngularJS, “ui-router” 和 “ui-jq”。

ui-router

ui-router 的 嵌套路由 和 嵌套视图, 请查看 官方文档

ui-load

ui-load 的 延时加载 js 和 css 文件。

angular.module('myModule', ['ui.load']);

uiLoad.load( ['path/style.css', 'path/script.js'] ).then(function() {
// 加载完成后执行。
}).catch(function() {
// 错误时执行。
});

ui-load 还可以使用在 ui-router 中，利用resolve:{} 来加载指令、控制器和服务。

ui-jq

ui-jq 调用元素上指定的jQuery函数或插件。 请查看 官方文档

ui-validate

ui-validate 验证指令可以很容易地创建自定义的验证表达式。 请查看 官方文档

指令

ui-toggle-class

切换到定义了指定 target 的元素。 <a ui-toggle-class="show" data-target=".nav"></a>

ui-shift

替换小屏幕制定了 target 的元素。 <div ui-shift="append" target="#nav"></div>

ui-scroll

滚动到定义了指定 target 的元素。 <a ui-scroll="target"></a>

ui-fullscreen

将元素以全屏的方式显示。 <a ui-fullscreen></a> 或 <a ui-fullscreen target="#target"></a>

jQuery插件

所有jQuery插件都依赖于ui-jq，不需要创建Angular指令。 你可以在js/config.lazyload.js中加入链接到 JQ_CONFIG, functionName => pluginPath, 将插件的js和css文件引入进来。
