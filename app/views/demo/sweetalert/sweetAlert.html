使用说明
  1. controller 注入服务 OperationService
  2. 参数说明
    • showConfirm: boolean, // 是否提示[调用后台接口前，以及操作成功的提示]，默认为 true
    • actionUrl: '', // 操作对应的后台接口[必填]
    • data: '', // 需要提交到后台的参数数据
    • isSuccessBack: true, // 操作成功是否返回
    • redirectState: 'app.home' // 操作成功的重定向地址(state跳转，isSuccessBack为false时起作用)
  3. 示例
    1. 调用接口前有提示，调用之后有提示，操作成功后停在当前页面 
      OperationService.interAction({
  'actionUrl': '/home',
  'data': { 'name': 'abc', 'age': 8 },
  'isSuccessBack': false
});
    2. 直接调用后台接口，成功不返回 
      OperationService.interAction({
	'showConfirm': false,
  'actionUrl': '/home',
  'isSuccessBack': false
});
    3. 操作成功返回[isSuccessBack 为 true时，redirectState 参数不起作用] 
      OperationService.interAction({
  'actionUrl': '/home',
  'isSuccessBack': true,
  'redirectState': 'app.home.contactlist2'
});
    4. 操作成功后重定向 
      OperationService.interAction({
  'actionUrl': '/home',
  'isSuccessBack': false,
  'redirectState': 'app.home.contactlist2'
});
    5. 提示框

      OperationService.alertAction('这里是提示信息');

      或者 OperationService.alertAction({alertMsg: '这里是提示信息'});
