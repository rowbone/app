弹出提示框
$scope.open = function() {
    ngDialog.open({
        template: 'deleteConfirm',
        controller: 'DeleteCtrl',
        closeByDocument: false
    });
};

可以在 DeleteCtrl 定义方法打开另外一个 dialog 对象

ngDialog.open({
	template: '<div class="text-success has-success"><span class="glyphicon glyphicon-ok"></span>删除成功</div>',
	closeByDocument: false,
	closeByEscape: false,
	plain: true,
	overlay: false
});
使用说明
  1. index.html中引入文件 
    <link rel="stylesheet" href="./vendor/ngDialog/css/ngDialog.css">
<link rel="stylesheet" href="./vendor/ngDialog/css/ngDialog-theme-default.css">
<link rel="stylesheet" href="./vendor/ngDialog/css/ngDialog-theme-plain.css">

<script src="./vendor/ngDialog/js/ngDialog.js"></script>
  2. app.js中注入依赖var app = angular.module('app', ['ngDialog']);
  3. 在controller中可直接注入 ngDialog 进行使用
参数说明
  1. template{String}
  2. plain{Boolean}
  3. controller{String} | {Array} | {Object}
  4. closeByDocument{}
  5. closeByEscape{}
  6. overlay{}

更多请参考项目主页：ngDialog on github
