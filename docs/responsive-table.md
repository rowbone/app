
## Angular responsive table   

#### 功能点：  
----------------- 

* 数据源[接口数据、json数据]：   
* 分页：   
* 单页显示条数：   
* 开发控制页面内容[列名、哪些列小屏收起或者不显示]：   
* 列控制可使用html：   
* 参数控制刷新：   
* 过滤搜索[前台、后台、单列搜索]：   
* 排序：   
* 单选返回：   
* 多选返回：   
* 点击触发回调方法：   
* 滚动到最上面head固定：
